# Inbox

Capture first. Sort later. Process weekly.

**Rule:** anything unplaced lands here. Don't let perfect placement block capture.

---

## How to process

1. Read each item
2. File to the right place (`context/`, `projects/`, `meetings/`, `log/`, `writing/`)
3. Delete from inbox once filed
4. Date: aim to clear weekly
